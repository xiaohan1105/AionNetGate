# AionGate 管理控制台

> 现代化、强大的游戏服务器管理界面

## ✨ 功能特性

### 🎛️ 实时监控仪表板
- **系统性能监控**
  - CPU 使用率实时图表
  - 内存使用情况
  - 磁盘使用统计
  - 网络流量监控

- **业务数据统计**
  - 在线人数实时显示
  - 24小时在线趋势图
  - 今日收入统计
  - 热门商品排行

- **实时日志流**
  - 彩色日志输出
  - 自动滚动
  - 暂停/继续
  - 日志过滤

### 👥 玩家管理
- **实时在线玩家**
  - 在线玩家实时标签展示
  - 玩家等级、位置信息
  - 一键踢出/封禁

- **玩家列表**
  - 高级搜索（账号/角色名/IP）
  - 状态筛选（在线/离线/封禁）
  - 批量操作（发送道具/点券/邮件）

- **快速操作**
  - 🔨 踢出玩家
  - 🚫 封禁账号（可设置时长）
  - 🎁 发送道具
  - 💰 发送点券
  - 📧 发送邮件

### 🛒 商城管理
- **商品管理**
  - 添加/编辑/删除商品
  - 价格设置（支持折扣）
  - 库存管理
  - 限购设置
  - 热卖/新品标记

- **销售数据**
  - 实时销售统计
  - 热门商品排行
  - 收入报表
  - 库存预警

- **快速调整**
  - 一键上架/下架
  - 批量价格调整
  - 库存快速补充

### 📢 公告管理
- **公告发布**
  - 富文本编辑器
  - 公告分类（系统/活动/维护/更新/新闻）
  - 置顶功能
  - 定时发布

- **全服公告**
  - 一键发送全服消息
  - 实时推送到所有在线玩家
  - 公告历史记录

### 📊 数据统计
- **可视化图表**
  - 在线人数趋势
  - 收入统计图
  - 玩家活跃度
  - 商品销售排行

- **数据报表**
  - 日报/周报/月报
  - 导出 Excel
  - 自定义统计周期

### 🔍 外挂检测
- **实时检测记录**
  - 进程检测
  - MD5 校验
  - 窗口类名检测

- **黑名单管理**
  - IP 黑名单
  - 硬件指纹黑名单
  - 自动封禁规则

- **检测日志**
  - 详细检测记录
  - 封禁历史
  - 统计分析

### ⚙️ 系统工具
- **快速操作**
  - 🚀 启动服务器
  - 🛑 停止服务器
  - 🔄 重启服务器
  - ⚠️ 维护模式切换

- **数据库工具**
  - 数据库备份
  - 数据恢复
  - SQL 执行器
  - 数据清理

- **配置管理**
  - 在线编辑配置
  - 配置版本管理
  - 一键重载配置

## 🎨 界面特色

### Modern Design
- 基于 ModernWPF UI 库
- Windows 11 设计语言
- 流畅的动画效果
- 深色/浅色主题切换

### 响应式布局
- 自适应窗口大小
- 智能卡片布局
- 优化的数据表格

### 直观操作
- 一键快速操作
- 拖拽式管理
- 右键快捷菜单
- 键盘快捷键支持

## 🚀 技术栈

### 框架
- **.NET 9.0** - 最新 .NET 版本
- **WPF** - Windows Presentation Foundation
- **ModernWPF** - 现代UI库

### 依赖库
- **LiveCharts** - 实时图表
- **CommunityToolkit.Mvvm** - MVVM 框架
- **RestSharp** - HTTP 客户端
- **Serilog** - 结构化日志
- **Dapper** - 数据库访问

## 📦 启动方式

### 开发环境
```bash
cd src/AionGate.Admin
dotnet run
```

### 发布
```bash
dotnet publish -c Release -r win-x64 --self-contained
```

## 🔧 配置

编辑 `appsettings.json`:

```json
{
  "ApiBaseUrl": "http://localhost:5000",
  "AutoRefreshInterval": 10,
  "LogLevel": "Information"
}
```
